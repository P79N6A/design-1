{{ $t('国际贸易出口流程及相关外贸术语') }}

国际贸易出口流程指的是每宗出口交易从询盘到实际的交货付款之间经过的具体的交易环节，还包括后续可能存在的纠纷仲裁以及索赔环节。

一般国际出口贸易流程(此流程以贸易术语CIF、结算方式TT等条件为基础)如下：

{{ $t('一、询价/询盘') }}

买家下单前，通常会通过邮件或者B2B平台做相应的Inquiry，询盘的内容通常包括产品型号、价格、数量以及交货期等。
买卖家结识的途径包括：国内外展会、专业B2B网站搜索、搜索引擎搜索如Google等.

{{ $t('二、报价') }}

出口企业业务员通常使用报价表回复客户询盘，确定货品名称、型号、数量、交货期、付款方式、包装方式及集装箱柜型等。
回复询盘的报价表（Quote Sheet）通常包含必要的货品信息，同时还包含必要的货品图片以及详细技术规格说明、是否支持定制（Customization）等.

{{ $t('三、客户下单') }}

经过充分洽谈和沟通后，买卖双方觉得价格和质量都满意(Order Confirmed)，下一步就是下P/O。
在实际业务中，订单可以以多种方式进行，可以是双方确认的合同，也可以是买方对卖方所开出的PI的回签确认,有时候也可以是L/C PI：proforma invoice即标明产品以及价格和数量的形式发票 P/O：purchase order

常用外贸订单付款方式:

1.L/C（Letter of Credit）信用证，一种可独立于合同和订单的完整契约文件,用于以银行为第三方为买卖方提供的交货付款契约文件。
2.T/T（Telegraphic Transfer ）T/T付款有多种，常用的为前TT方式：30% Deposit，70% against B/L Copy

{{ $t('四、按单生产') }}

出口制造企业安排生产是履行合同非常关键的环节。
1.在L/C付款方式下，一般需要待收到买方开具的L/C并审核无误后，方可根据订单安排生产。对于信誉较好的买方客户，有依据证明客户肯定会履行付款义务，也可现行安排生产
2.T/T付款方式下，收到预付30%定金，即可安排生产，同样对于信誉好的客户也可灵活对待，提前安排生产用以确保交期，再行催款。
来单生产：BTO，Build To Order

{{ $t('五、验货') }}

卖方根据订单或者L/C的规定按时、按质、按量的准备交易标的物，备货完成后，装柜前，买方一般要求进行验货，以确保产品质量和规格。买方通常委托第三方验货公司进行验货，验货完成后，出具验货报告（Inspection Report）。建立稳定的信任关系后，有些买方只要求制造企业出具自身验货报告即可。

{{ $t('六、商检') }}

根据《中华人民共和国进出口商品检验法》规定，一切进出口商品都必须经过检验，未经过国家商检机构检验或者检验不合格，不准出口。
属于法定检验的商品，或者合同规定由国家商检机构检验出证的商品，在货物备齐后，应向商品检验局申请检验，一般应在商品出运前至少一周内提出申请，报验时应填写“出口报验申请单”，并随附合同，以及出口货物报关单等通关用的凭证。
经检验合格，由商检机构签发检验证书，或者“出境货物通关单”（商检证）。对于生产地（商检地）与出境地（报关地）不同时，需先从生产地的商检部门取得取证凭条，然后于报关前在报关地的商检部门换取“出境货物通关单”，才可以顺利报关。
验货和商检的区别，一个是客户行为，目的是在出货前验证订单是否已得到如量如质地执行，一个是官方行为，由出入境检验检疫机构接受检验，检验合格，签发出境货物通关证。

出境货物通关单：Export Customs Clearance Bill

{{ $t('七、租船定舱') }}

这是货物交运环节，办理国际货物运输的手续。

FOB=Free on Board,离岸价格，装运港船上交货 CFR=Cost and Freight,成本+运费，由卖方负责运费 CIF=Cost Insurance and Freight,成本+运费+保险费，卖方负责运费并支付保费。
当交易是按照CFR或者C.I.F.价格成交时，租船订舱手续由卖方负责办理；如果FOB价格条件成交时，租船订舱由买方负责办理，此情况下，通常客户会指定某一运输代理公司或者船运公司。
业务员通常在交货期两周前向货代公司发出托运单，通常在开船一周前可拿到货代发来的订舱确认单，注明船名，航次等信息。

托运单：Booking Form 订舱确认单：Booking Confirmation

{{ $t('八、结算') }}

在交易磋商时，买卖双方基于对现实利益的考量和对风险的评估，在结算方式上达成一致的意见，并在合同中明确规定。通常进出口企业主要采用三种传统的货款结算方式为：汇付、托收和信用证。
最近一二十年，一种新兴的国际结算方式-国际保理，迅速发展，逐渐和托收、信用证成为国际贸易新的三大结算方式。

1.汇付：Remittance 2.托收：Collection 3.信用证：L/C 4.国际保理：International Factoring

随着在线交易的兴起，增加了在线支付方式，如paypal，ipay等支付方式。
汇付(Remittance)是买方通过银行将货款汇给卖方，其主要表现方式是赊销和预付款.

赊销：Open Account，O/A 预付款：deposit in advance,balance before delivery,如前TT方式

托收(Collection)是卖方在货物出运并开具汇票之后，连同全套货运单据委托出口地银行通过它在卖方所在地银行或者代理向买方收取货款。托收有付款交单和承兑交单两种。 付款交单(D/P)：Documents against Payment 承兑交单(D/A): Documents against Acceptance。
信用证(L/C)是应买方要求，由开证银行开出，以买方为受益人的一种保证付款凭证。
国际保理又称为承购应收账款，指在以商业信用出口货物时（如以D/A作为付款方式)，出口商/卖方交货后把应收账款的发票和装运单据转让给保理商，即可取得应收取的大部分贷款，日后一旦发生买方不付或者逾期，则由保理商承担付款责任。国际保理是银行作为保理商为国际贸易几张赊销方式(O/A)提供出口贸易融资、销售账务处理、收取应收账款及买方信用担保何为一体的综合性金融服务。

常用结算方式：
1.30%T/T 预付款 + 70%装柜前付款
2.30%T/T 预付款 + 70%即期、保兑、不可撤销信用证
3. 即期、保兑、不可撤销信用证+CIF运输条款
4.即期、保兑、不可撤销信用证+FOB运输条款
5.30% TT预付款 + 70%见提单传真件付款（30%后T/T），适用于小额贸易

最新结算趋势：根据最新中国外汇的统计，我国对外贸易中，传统的信用证结算比例已经下降到了20%以下，而赊销比例则上升到了70%以上。欧美国际贸易信用证使用更是下降到了10%以下，赊销比例上升到90%左右。

{{ $t('九、投保') }}

这里指的是办理国际运输的保险手续。
当交易按照C.I.F.价格条件成交时，应由卖方办理。此处注意：投保人在确定投保险别和投保金额时均按照信用证的规定办理。
当交易按照F.O.B 或者C.F.R.价格条件成交时，应由买方办理。
货运保单：Cargo Transportation Insurance Policy 产品责任险保单：Products Liability Insurance Policy

{{ $t('十、拖柜装货') }}

确定船期后，待货备妥，由卖方企业的船务员提供内容清晰完整的“订舱确认单”给委托的拖车公司，由拖车公司前往指定地点提柜，然后回工厂装货，并运往指定的交货码头，以便下一步办理出口报关和装船。

{{ $t('十一、报关') }}

货柜到达码头后，在实际装船之前，必须要向海关进行申报。
申报时需要交验货物报关单、贸易合同、商业发票及装箱单等重要单证。海关核对查验无误，在装货单上签章放行后，货物才能装船。

货物报关单：Customs Declaration Form
贸易合同： Sales Contract
商业发票：Commercial Invoice
装箱单：Packing List

{{ $t('十二、获取提单') }}

提单：Bill of Lading，一般三正本三副本
获取时间：货物通关装船出运后

国际运输中最重要的文件就是作为货权凭证的提单（B/L），卖方船务员在与船运公司或者货代确认好提单内容，并支付相应的运杂费后，即可从船运公司或者货代处获得提单。

一般在开船两天内，船务员要将提单补料内容（指出现在提单上的内容信息）传真给船运公司或者货代，用以制作准确的提单。船运公司或者货代在提供正本提单之前，会先提供提单样板工船务员确认。

提单补料：Shipping Instruction
正本提单：Original B/L
提单副本：B/L Copy
全套提单：Full Set B/L

{{ $t('十三、交单议付') }}

采用T/T付款方式，卖方在取得提单后即将其传真给买方，买方据此B/L Copy付尾款，确认收到尾款后，卖方将提单正本及其他文件寄给买方，买方拿到提单后即可在目的地港办理提货。

信用证(L/C)：Letter of Credit
采用L/C收汇的应在规定的交单时间内，备齐经过严格审核的全套单据，确保没有错误，提交银行议附。 提交银行议附的单证需要严格按照L/C的要求备齐，重要单证包括:商业发票、装箱单、产地证（普惠制产地证）、提单等。

普惠制产地证： Generalized Systems of Preferences Certificate of Origin

{{ $t('十四、出口外汇核销') }}

出口核销制度指的是货物出口后，由外汇管理部门对相应的外汇进行核销。
出口收汇核销管理是指外汇局在商务、海关、税务、银行等有关部门配合协助下，以出口货物的价值为标准核对是否有相应的外汇收回国内的一种事后监管措施，是对出口收汇贸易真实性的审核。
出口企业在货物报关后，凭海关盖章放行报关单的核销联以及结汇时银行签章的核销单、结汇水单、发票等到当地外汇局办理出口收汇核销手续，证实已按额收回货款。
报关单：Customs Declaration Form，一般为一式六联：海关作业联、海关留存联、企业留存联、海关核销联、出口收汇用的证明联、出口退税用的证明联。

{{ $t('十五、出口退税') }}

国家为了鼓励我国出口货物在国际市场上参与公平竞争的一种税收措施，对出口产品退还或者免征其在国内依税法缴纳的增值税和消费税。

出口企业在办理外汇核销后，填报《出口货物退(免)税申报》并提供办理出口退税的有关凭证，向税务局涉外分局申请退税。 办理退税的凭证包括： 1.盖有海关验讫章的《出口货物报关单》出口退税联
2.核销单（外汇管理局退回的出口退税专用联）
3.出口销售发票
4.结汇水单或者收汇通知书

{{ $t('十六、贸易仲裁') }}

仲裁是解决对外贸易争议的一种方式。当买卖双方在发生争议的，可以采取友好协商、仲裁和司法诉讼三种不同的方式解决。当有些争议无法通过友好协商解决是，就不得不将争议提交第三方商事仲裁机构进行仲裁，该机构作出的裁决为终局裁决，对双方均具有约束力。

协商：Negotiation
争议：Dispute
仲裁条款：Arbitration Clause
仲裁委员会：Arbitration Commission
赔偿：Compensation

{{ $t('十七、索赔') }}

在进出口交易中，当被保险货物遭受损失或当一方违反合同规定直接或者间接地给另一方造成损失时，受损失一方可以向保险人或违约方提出赔偿要求。如果双方就索赔条件意见不一，无法协调解决时，就可依据合同仲裁条款规定，将争议提交第三方仲裁

理赔：settlement of claims
检验报告：survey report
次品：inferior goods
不合格品：disqualified goods

至此，出口贸易的基本流程基本完结。
